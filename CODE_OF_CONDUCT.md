# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be respectful and constructive.
- Disagree on technical merit, not personal terms.
- No harassment, discrimination, or hostility of any kind.

Reports: open a private issue or email `victorgutierrezgomes@gmail.com`. The maintainer will respond within 7 days.
